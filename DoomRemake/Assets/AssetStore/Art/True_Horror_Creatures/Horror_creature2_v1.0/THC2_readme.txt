NighMare Creature #2 readme.

Controls:

To change the combat and normal mode press 1 or 2
(most actions available in combat mode)
2 - turn on runing mode  ( to off press 1 );
3 - sleep mode

moving - directional arrows

roar - c
bite - x
attack1 - mouse 1
attack2 - mouse 2
attack3 - mose 3
hit - u
die/fall - i
v - rise up ( only after falling )
jump - space
