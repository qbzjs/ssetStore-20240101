NighMare Creature #5 readme.

Controls:

To change the mode press:
1 - still mode
2 - combat mode
(most actions available in the combat mode)

moving - directional arrows

mouse 1 - attack1
mouse 2 - attack2
mouse 3 - attack3

x - laugh
c - roar
u - hit
i - death

o - death or falling
l - rise up
