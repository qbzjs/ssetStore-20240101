NighMare Creature #4 readme.

Controls:

To change the mode press:
1 - still mode
2 - combat mode
3 - down mode
(most actions available in the combat and the down modes)

moving - directional arrows

mouse 1 - attack1
mouse 2 - attack2
mouse 3 - attack3
space - jump
x - bite
c - roar (mode 1 and 2), eating (mode 3)
u - hit
i - death
