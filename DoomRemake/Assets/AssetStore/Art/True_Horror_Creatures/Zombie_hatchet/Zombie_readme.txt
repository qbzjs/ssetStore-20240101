Zombie Hathcet readme.

Controls:

To change the mode press:
1 - still mode
2 - still mode another walk mode
3 - combat mode + run
(most actions available in the combat mode)

moving - directional arrows

mouse 1 - attack1
mouse 2 - attack2
mouse 3 - attack3

p - defence ( with hatchet )
c - laugh
u - hit
y - power hit and drop hatchet
i - falling (drop hathet if armed)
k - rise head ( going to lie mode )
l - rise up
o - death
